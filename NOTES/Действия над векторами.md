# Скалярное произведение
Скалярное произведение можно посчитать так
Представим что у нас есть вектора
$\vec{a}=\{a_1, a_2, \text{ ... }, a_n\}$
и
$\vec{b}=\{b_1, b_2, \text{ ... }, b_n\}$
Тогда их скалярное произведение будет равно
$$\vec{a} \cdot \vec{b}=a_1\cdot b_1 + a_2\cdot b_2 +\text{ ... } + a_n\cdot b_n$$

Однако есть небольшая проблема при работе с комплексными числами. Потому что если перемножать два комплексных вектора то у нас результат будет сложным для интерпретации и может не иметь физического смысла. Например, если мы умножим два комплексных числа, результат будет другим комплексным числом, и его модуль (длина) не всегда будет положительным. Но если один из векторов будет комплексно сопряжённым то всё будэ гуд

$$\vec{a} \cdot \vec{b^*}$$
