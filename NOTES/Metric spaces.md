# Что это?
Метрические системы(metric spaces) — это системы, которые определяют способ измерения расстояния между точками в пространстве.

Metric - это математическая функция, которая определяет расстояние между двумя точками в некоем set. Формально метрика для set X - это функция d: $X \times X \to \mathbb{R}$. То есть метрика это функция, которая принимает две точки из множества XX и возвращает действительное число (расстояние между этими точками). $X \times X$ означает декартово произведение множества X на себя, т.е. множество всех возможных пар точек (x,y), где x и y принадлежат X.

Чтобы пара (metric, set) т.е. (d, X) были metric space нужно чтобы выполнялись следующие аксиомы:

1. Неотрицательность: $d(x, y) \ge 0$. Расстояние всегда неотрицательно
2. $d(x, y) = 0, \ \iff x=y$. Читается расстояние между точками x и y равна нулю, тогда и только тогда, когда x=y.
3. Симметричность: $d(x,y)=d(y,x)$. Расстояние от xx до yy такое же, как и от y до x.
4. [[Triangle inequality]]: $d(x,y) \le d(x,y)+d(y,z)$. Расстояние напрямую из x в z не больше, чем расстояние через любую другую точку y.
