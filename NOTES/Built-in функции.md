# built-in функции
В компиляторе gcc есть свои встроенные функции которые могут оказаться полезными, особенно если мы работаем с битами отдельно. По сему они особенно полезны при работе с микроконтроллерами. В этой заметке будут приведены наиболее полезные(как мне кажется) built-in функции

## `int __builtin_parity(uint x)`
Возвращает чётность числа `uint x`
То есть если сумма 1 в двоичном представлении числа будет чётным числом, то он вернёт 0, в ином случае 1. 

## `int __builtin_popcount (uint x)`
Возвращает количество 1 в двоичном представлении числа x
