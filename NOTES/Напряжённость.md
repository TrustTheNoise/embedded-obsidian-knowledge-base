# Напряжённость(H)
**Напряжённость**(Н) - это характеристика магнитного поля, которая показывает насколько сильно поле действует на магнитные объекты
То есть чем больше напряжённость магнитного поля в какой-то точке тем больше оно притягивает/отталкивает другой магнит
**Напряжённость** показывает с какой силой поле будет действовать на проводник с током 1А. При этом (а это важно!) дело должно происходить в вакууме. То есть напряжённость это личная характеристика магнита независимая от окружения
$1\frac{А}{м}$ - магнитное поле с такой напряжённостью появляется на расстоянии 16 см от проводника, по которому идёт ток 1А.
Напряжённость определяется в какой-то точке относительно магнита вызывающего магнитное поле
$$\color{red}{H=\frac{I}{2 \pi r}}$$
