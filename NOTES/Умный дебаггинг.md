# Что было сделано
Я сделал что-то типо около исследования по тому сколько нужно тактов на различные процессы в микроконтроллере
[Замер времени выполнения](https://docs.google.com/document/d/1zIEqUaiV4jS2FQ8t5xlDj1zX8u0pakN2FC9pKzicTKI/edit?tab=t.0#heading=h.h552bbvrb6oa)
Также я узнал что лучшим вариантом для того чтобы точно узнать количество тактов пройденных кодом до этого момента можно использовать 
DWT->CYCCNT
Чтобы можно было использовать этот код нужно настроить таким образом
```c
CoreDebug->DEMCR |= CoreDebug_DEMCR_TRCENA_Msk;
DWT->CYCCNT = 0;
DWT->CTRL |= DWT_CTRL_CYCCNTENA_Msk;
```
