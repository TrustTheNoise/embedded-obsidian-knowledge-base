# Разделение

Цифровые фильтры делятся на две фундаментальные группы, различающиеся как по внутренней структуре, так и по динамическим свойствам выходного сигнала:

- КИХ фильтры

Их импульсная характеристика строго конечна: ограниченное число отсчётов, после которых отклик фильтра точно равен нулю. КИХ фильтры всегда линейно-фазовые при симметрии коэффициентов, а также ВСЕГДА устойчивы.

Также для КИХ фильтров есть множество различных способов получить различные типы фильтров, НЧ, ВЧ, полосный и режекторные. Самый простой из них при помощи НЧ фильтров получить другие виды фильтров. Об этом рассказано в [[КИХ-фильтры]]. Также есть возможность получать КИХ фильтры напрямую из нужной АЧХ через обратное преобразование Фурье для получения бесконечной импульсной характеристики и затем использования окон для уменьшения отсчётов импульсной характеристики

КИХ фильтры обычно требуют большего порядка для получения узкой полосы или крутой АЧХ, чем БИХ-фильтры;

- БИХ фильтры

Импульсная характеристика теоретически бесконечна из-за наличия обратной связи. 

БИХ фильтры намного компактнее: в десятки раз меньший порядок по сравнению с КИХ для той же крутизны фильтра; Хотя КИХ фильтры не обеспечивают точной линейности фазы и требуют анализа устойчивости.

Обычно проектируются через аналоговые прототипы (Баттерворт, Чебышёв, эллиптические), затем преобразуются в цифровую область (bilinear transform).
