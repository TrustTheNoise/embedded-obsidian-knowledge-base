
# Функции

Значения по умолчанию для аргументов функций вычисляются в момент определения функции, а не в момент её вызова. Это может вызывать проблемы при использовании **изменяемых типов** (например, словаря или списка) и их модификации внутри тела функции, поскольку внесённые изменения **сохраняются между вызовами функции**.

```py

def add_to_dict(args={'a': 1, 'b': 2}):
    for i in args.keys():
        args[i] += 1
    print(args)
    

add_to_dict()
# {'a': 2, 'b': 3}
add_to_dict()
# {'a': 3, 'b': 4}
add_to_dict()
{'a': 4, 'b': 5}
```
