# Что это?
Linear equation - это алгебраическое уравнение, в котором каждая переменная имеет степень не выше первой. Это означает, что переменные в линейном уравнении не умножаются друг на друга и не возводятся в степень. Общий вид линейного уравнения для 2-dimension plane представляет из себя

$$ax+by+c=0$$
Где a, b, c - коэффициенты, а x и y - переменные.

# Algebraic description of linear equation

## Simple linear equation
Для того чтобы решить простейший linear equation нам нужно просто найти значение неизвестного. Для этого нам нужно привести к простейшему виду $ax=b$. А для этого можно воспользоваться 2 методами которые не изменяют решение уравнения. А именно
1. Прибавление(разность) обеих частей уравнения на одинаковое значение
2. Умножение(деление) обеих частей уравнение на одинаковое значение <span style="color:red">отличное от нуля!</span>

Например
$$2x+5=x-1$$
Отнимем 5 из каждой части
$$2x=x-6$$
Отнимем x из каждой части
$$x=-6$$
Вот и ответ >;3

## Linear equation with absolute value

Как понятно из названия в таких equation будет какое либо absolute value что не даёт нам просто найти значение для неизвестного, как в обычном linear equation. Ведь функция absolute value которая геометрически определяет расстояние между двумя точками выглядит таким образом

$$
|x|
=
\begin{cases}
	x, \text{ if } x \ge 0 \\
	-x, \text{ if } x < 0
\end{cases}
$$

Соответственно в случае equation with absolute value нам нужно рассматривать несколько случаев для неизвестного. Можно решать геометрически(разбирая какие x будут иметь нужное расстояние), а также алгебраически. Чтобы решить алгебраически нужно найти turning point для этого absolute value. Т.е. точка которая граничит возможные варианты для неизвестного в absolute value. Лучше объяснить это на примерах

### Пример 1
$$|x-1|=2$$
Итак нам нужно найти turning point для $|x-1|$, то есть такое значение x которое граничит с двумя условиями для absolute value. Граничным условием для absolute value $|x|$ всегда является $x=0$, потому что при x=0 функция $|x|$ достигает своего минимального значения и меняет направление с -x на x.

Соответственно turning point для $|x-1|$ будет
$$x-1=0 \to x=1$$
Тогда у нас есть два случая.
- Случай 1:
	$x \ge 1$, в таком случае $|x-1|=x-1$, тогда уравнение решается так:
	$x-1=2 \to x=3$
- Случай 2:
	$x < 1$, в таком случае $|x-1|=-x+1$, тогда уравнение решается так:
	$-x+1=2 \to x=-1$

Таким образом у данного equation решение это set $\{-1, 3\}$
Это решение можно найти геометрически. Геометрически equation with absolute value $|x-1|=2$ это нахождение такого x при котором расстояние от x до точки 1 будет равно 2. Если посмотреть на числовую прямую, то такие x и будут -1 и 3

### Пример 2
$$|x-1|+|x-5|=2$$
Итак у нас тут уже 2 функции absolute value. Тогда получается у нас есть 2 turning point, а именно
$x=1 \land x=5$
Рассматриваемых случаев для absolute value с 1 неизвестной всегда на 1 больше чем turning points. Значит нам нужно рассмотреть 3 случая.
![[Pasted image 20250701122142.png|400]]

Рассчитаем уравнение для всех трёх вариантов
- <span style="color: green">Случай 1:</span>
	$-x+1-x+5=2$
	$-2x=-4$
	$\cancel{x=2}$ - false root, т.к. случай 1 рассматривает $x<1$, а у нас получился root x=2
- <span style="color: purple">Случай 2:</span>
	$x-1-x+5=2$
	$4=2$ - inconsistent
- <span style="color: pink">Случай 3:</span>
	$x-1+x-5=2$
	$2x=8$
	$x=4$

Таким образом решением будет только число 4

# Geometric description of linear equation

Equation нужны не только для алгебраического решения(нахождения неизвестного x), но также при помощи уравнений можно математически описывать различные объекты на графике. В этой главе и будет рассказано какие объекты могут быть описаны через linear equation. По факту же через linear equation могут быть описаны только простейшие линии.

## Equation прямой линии

Equation для прямой линии выглядит вот так:
$$y=mx+b$$
Где **m** называется угловым коэффициентом, или наклоном(slope), **b** называется "точкой пересечением с осью oy" в английской терминологии используется слово "перехват"(intercept) как если бы ось oy перехватывала линию из одной половины в другую.

Slope - это число, которое показывает, насколько круто прямая поднимается или опускается. Он определяет угол наклона прямой относительно положительного направления оси ox. Соответственно slope m может определяться как отношение изменения значения y к изменению значения x между двумя точками на прямой:

$$m=\frac{\Delta y}{\Delta x} = \frac{y_2-y_1}{x_2-x_1} = \frac{y_1-y_2}{x_1-x_2}$$

В этой формуле $\Delta y$ называют rise(подъём) и указывает на вертикальное изменение. А $\Delta x$ называют run(перемещение) и указывает на горизонтальное изменение

Физический смысл для slope такой: 
- Если m>0, прямая поднимается вверх слева направо, что означает положительный наклон.
- Если m<0, прямая опускается вниз слева направо, что означает отрицательный наклон.
- Если m=0, прямая горизонтальна и не имеет наклона.

Intercept - это точка, в которой прямая пересекает ось oy. То есть это точка когда x=0. То есть это точка (0,b).

### Построение уравнения для прямой линии

#### Если известны две точки
Если нам известны две точки через которые проходит наша прямая, то для нахождение slope можно подставить просто координаты в формулу для slope

Допустим у нас есть две точки (-3, 4) и (1, -4). Подставим координаты в формулу для slope

$$m=\frac{\Delta y}{\Delta x} = \frac{y_2-y_1}{x_2-x_1} = \frac{-4-4}{1-(-3)}=-\frac{8}{4}=-2$$

Для того чтобы найти y-intercept(значение b) можно подставить в формулу y=mx+b с известным m координаты одной из известной точки. А затем просто решить уравнение через b. Возьмём точку (-3,4)
$4=-3m+b$
$4=-3\cdot(-2)+b$
$b=-2$

В итоге мы получаем уравнение для прямой:
$$y=-2x - 2$$
