# Цифровой шум

В цифровой обработке сигналов существует потребность в генерации шумов различных типов. Такая необходимость возникает при тестировании алгоритмов, которые должны оставаться работоспособными в присутствии шума заданного уровня. Генерация цифрового шума строится на базе генератора случайных чисел. В большинстве языков программирования для выполнения этой задачи предусмотрена специальная функция - функция рандомайзера. Случайные значения полученные через эти функции имеют равную вероятность появления в пределах границ выставленных в функций(например в Си функция `rand()` выдаёт числа от 0 до RAND_MAX)  

> [!summary] 
> То есть функция плотности вероятности для такой величины является константой на заданном отрезке [a,b] и равна нулю $\frac{1}{b-a}$, при этом она равна нулю вне этого отрезка
> Поскольку отсчеты такого сигнала независимы и имеют равномерную плотность вероятности, такой шум называют **равномерным белым шумом**

Например если используя такой код в си получить сигнал из случайных чисел
```c
x[i] = rand() / RAND_MAX;
```
То мы получим сигнал со случайными значениями расположенными в диапазоне от 0 до 1 и иметь равную вероятность появления в пределах этих границ. На рис а) ниже показан такой сигнал сформированный из 128 отсчётов.
![[Pasted image 20260310150336.png|800]]



Среднее значение исходного процесса при генерации данного сигнала составляет 0.5. А вот насчёт СКО интересный момент. Смотри расчёт среднего значения можно произвести двумя способами.
1. Обычное среднее, если у нас две 3 и три 4, тогда мы считаем среднее таким образом
$$\mu = \frac{3+3+4+4+4}{5}$$
2. Но можно посчитать это иначе, через "вес"(долю) каждого числа
$$\mu = 3 \cdot (\frac{2}{5}) + 4 \cdot (\frac{3}{5})$$

Как раз таки функцией доли каждого из числа и есть функция распределения вероятностей, соответственно если мы знаем функцию распределения вероятностей $f(x)$, то мы можем посчитать и среднее значение как:
$$\mu = \int_J x \cdot f(x) dx$$
где x - значения сигнала.

Равномерное распределение вероятностей должна иметь площадь 1, и она имеет форму квадрата, поэтому можно просто найти формулу функции $f(x)$ через формулу площади квадрата, где отрезок $(b-a)$ - сторона прямоугольника, $f(x)$ - высота прямоугольника
$$(b-a) \cdot f(x) = 1 \implies f(x) = \frac{1}{b-a}$$

Тогда чтобы найти СКО, нам нужно найти среднее значение которое равно:


![[Pasted image 20260310150358.png]]
Как мы видим из графика если сложить 12 раз рандомное число то мы получим цифровой шум, чей график функции плотности будет почти идентичен гауссовой функции.
Это первый метод как получить нормально распределённый сигнал. Для этого нам нужно:
1) сложить 12 случайных чисел; 
2) из полученной суммы вычесть шесть, делая среднее значение равным нулю; 
3) умножить значение отсчёта на величину желаемого СКО; 
4) добавить требуемое среднее значение. 

![[Pasted image 20240429214135.png]]
Как и ранее, с помощью данного подхода можно генерировать нормально распределённые случайные сигналы с произвольным средним значением и произвольным СКО. Для этого возьмите каждое число, полученное с помощью данного выражения, умножьте на желаемое значение СКО и добавьте требуемое среднее значение.
