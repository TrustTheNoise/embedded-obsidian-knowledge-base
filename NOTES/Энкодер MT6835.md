# Энкодер MT6835
Поворотный датчик положения MagnTek MT6835 представляет собой микросхему, основанную на передовой технологии AMR. Вращающееся магнитное поле в полосе движения датчика x-y выдает два выходных сигнала синусоидальной формы, которые указывают угол (α) между датчиком и направлением магнитного поля. Датчик чувствителен только к направлению магнитного поля в плоскости x-y, поскольку выходной сигнал чувствительного элемента специально сконструирован таким образом, чтобы не зависеть от напряженности магнитного поля. Это позволяет устройству быть менее чувствительным к колебаниям магнитных полей, рассеянному магнитному полю, изменению воздушного зазора и смещению оси.

Режим инкрементного вывода ABZ является чип доступен в этой серии датчиков, что делает его пригодным для замены различных оптических датчиков. Максимальное разрешение составляет 16 384 импульса или 65 536 шагов за оборот 
4-проводной интерфейс SPI позволяет главному микроконтроллеру считывать 21-разрядные данные об абсолютном угловом положении с MT6835. 
Абсолютное угловое положение также выводитсяв виде 12-разрядного ШИМ-выхода.

## Как перевести 21 битный Angle в абсолютные градусы 0-360
![[Pasted image 20241011220931.png]]
В коде эта формула будет выглядеть так:
```c
double convertAngleToDegrees(uint32_t angle) {
    // Маска для получения 21 бита
    uint32_t angle21bit = angle & 0x1FFFFF;  // 21-битное значение угла
    // Преобразование в градусы
    double degrees = ((double)angle21bit / (double)(1 << 21)) * 360.0;
    return degrees;
}
```
> [!warning] 
> Смотри не запутайся в числителе в формуле, это и есть десятичное представление angle, то есть если ты уже засунешь в 1 переменную uint32_t все эти 21 бит, то в десятичном виде(который основной в си) у нас и будет уже числитель из формулы
