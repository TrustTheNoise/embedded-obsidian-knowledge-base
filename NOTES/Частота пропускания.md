# Частота пропускания
Частота пропускания - это диапазон частот, в котором сигнал может проходить через систему без существенного искажения. Этот диапазон определяется нижней и верхней граничными частотами, между которыми система может передавать сигнал с достаточной точностью.

Например, если мы говорим по телефону, наш голос преобразуется в электрический сигнал, который передается через телефонную линию. Эта линия имеет определенную частоту пропускания, которая ограничивает диапазон частот, которые могут быть переданы через нее.
