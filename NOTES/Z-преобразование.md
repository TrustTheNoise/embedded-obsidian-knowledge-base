# Z преобразования
Z-преобразования - это буквально [преобразование Лапласа](Преобразование%20Лапласа) но для дискретных систем
$$L\{ x(t) \}=\int_{0^-}^{\infty}x(t)e^{-st}dt$$
Только теперь нам нужен дискретный сигнал $x(t) \xrightarrow{t=nT_s, n\in\mathbb Z} x(nT_s)=x[n]$
Тогда наше преобразование Лапласа меняется в $\sum_{n=0}^{\infty}x[n]e^{-snT_s}$ ну так как у нас дискретный сигнал то интеграл заменяется на сумму
Теперь заменим $(\underbrace{e^{sT_s}}_{Z})^{-n}=Z^{-n}$ тогда мы и получим Z-преобразование
$$\large \color{red} Z\{x[n] \}=\sum_{n=0}^{\infty}x[n]z^{-n}=X(z)$$

= 
$$

$$
= 
	z^{-M}( 
	\underbrace
		{ 
			\sum_{k=0}^{\infty} x[k]z^{-k} 
		}
		_
		{
			Z\{x[k]\}
		} 
	+
	\underbrace
		{

		\color{orange}\text{начальные условия}

\xrightarrow{x[k]=0,\ k<0}

То что я подписал как начальные условия это не совсем точно. Да это является эквивалентом начальным условиям в [преобразовании Лапласа](Преобразование%20Лапласа#Свойства%20преобразования%20Лапласа). Однако тут это не значение сигнала и его производных во время t=0, а значение сигнала в отрицательные отсчёты. А в реальности мы считаем что сигнал в отрицательных отсчётах равен 0(что логично мы смотрим на сигнал начиная с точки 0, а отрицательные значения нам не важны) по сему мы отбрасываем эти "начальные условия"
Вообще в целом для дискретных сигналов $x[n-M]$ это эквивалент производной $f^n(t)$ для непрерывных сигналов

Мы разобрали эквивалент производной для дискретных сигналов, теперь нужно рассмотреть как будет себя вести z преобразование с эквивалентом интегрирования
$$
	{\large \color{red} Z\{ x[n+M] \} }
=
	\sum_{n=0}^{\infty} x[ \underbrace{n+M}_{k} ] z^{-n}
$$
$$
=
	\sum_{n=M}^{\infty} x[k]z^{-k+M}
\xrightarrow{x[k]=0 \text{ при } k=0,...,M-1}
	z^M\sum_{n=0}^{\infty} x[k] z^{-k}
=
	\large \color{red} z^M X\{ z \}
$$

Рассмотрим другие свойства

$${\color{red} Z\{a^n x[n]\} } = \sum_{n=0}^{\infty}x[n](a^{-1}z)^{-n}=Z\{x[n] \}(a^{-1}z)={\color{red} X(\frac{z}{a}) }$$
Т.е. если мы умножаем сигнал на геометрическую последовательность $a^n$, тогда мы в z-изображении делим z на a, таким образом мы можем увеличить масштаб при a<1, и уменьшать масштаб при a>1, это называется <span style="color: orange">свойство масштабируемости</span>

# Пример
Рассмотрим такой [IIR фильтр](Фильтрация%20и%20свёртка#IIR%20фильтр) 
![[Pasted image 20250126124513.png]]
Подадим на вход $\Large x[n]=(\frac{1}{4})^n 1[n]\xrightarrow{Z}\frac{1}{1-\frac{1}{4}z^{-1}}$
Его difference equation
$$y[n]+\frac{1}{2}y[n-1]=x[n]-x[n-1]$$
Переведём в Z изображение
$$Y(z)+\frac{1}{2}z^{-1}Y(z) = X(z)-z^{-1}X(z)$$
$$Y(z)=\underbrace{\frac{1-z^{-1}}{1+\frac{1}{2} z^{-1}}}_{W(z)} X(z)$$
W(z) - передаточная функция но для дискретных систем
Но нам нужно найти какой будет выход у системы
$$Y(z)=\frac{1-z^{-1}}{1+\frac{1}{2} z^{-1}} \cdot \frac{1}{1-\frac{1}{4}z^{-1}}$$
Теперь нам нужно получить обратное z преобразование. Но так просто это сделать не получится. Нужно преобразовывать эту функцию
Можно конечно убрать эти $z^{-1}$
$$
	\frac{1-z^{-1}}{1+\frac{1}{2} z^{-1}} 
\cdot 
	\frac{1}{1-\frac{1}{4}z^{-1}}
\cdot
	\frac{z^2}{z^2}
=
	\frac 
		{ (z-1)z } 
	{
		(z+\frac{1}{2})(z-\frac 1 4)
	}
$$
Но такое решение будет куда сложнее по итогу. В этом случае ещё ладно, но ведь в знаменателе могут быть полиномы больших порядков... Поэтому мы не ебём себе мозги и просто используем $z^{-1}$, если так вам проще то мысленно делайте замену $z^{-1}=q$
В таком случае мы  просто можем использовать метод разложения на простые дроби

$$
	\frac { 1-z^{-1} } 
	{
		(1+\frac{1}{2} z^{-1}) (1-\frac{1}{4}z^{-1}) 
	} 
= 
	\frac
	{ 
		\overbrace{A}^{z^{-1}=-2, \ \frac {1+2} { 1+\frac {1}{4} 2} } 
	}
	{ 1+\frac{1}{2} z^{-1} }
	+
	\frac
	{
		\overbrace{B}
		^
		{ 
			z^{-1}=4, \ \frac{1-4}{ 1+\frac{1}{2}4 } 
		}
	}
	{
		1-\frac{1}{4}z^{-1}
	}
=
	\frac{2} { 1+\frac{1}{2} z^{-1} } 
	+ 
	\frac{-1} {1-\frac{1}{4}z^{-1} }
$$

# Передаточная функция дискретной системы
Найдём общее решение для IIR 2-го порядка c нулевыми начальными значениями
![[Pasted image 20250126132847.png]]
$$
	a_0y[n] + a_1y[n-1] + a_2y[n-2]
=	
	b_0x[n] + b_1x[n-1] + b_2x[n-2]
$$
Возьмём z преобразование
$$
	a_0Y(z) + a_1z^{-1}Y(z) + a_2z^{-2}Y(z)
=	
	b_0X(z) + b_1z^{-1}X(z) + b_2z^{-2}X(z)
$$
Передаточная функция у такой системы будет
$$
\color{red} \large
	H(z)
=
	\underbrace
	{
		\frac
		{
			b_0 + b_1z^{-1} + b_2z^{-2}
		}
		{
			a_0 + a_1z^{-1} + a_2z^{-2}
		}
	}
	_
	{
		\text{для решения дифуров}
	}
=
		\underbrace
	{
		\frac
		{
			b_0z^2 + b_1z + b_2
		}
		{
			a_0z^2 + a_1z + a_2
		}
	}
	_
	{
		\text{для нахождения полюсов и нулей}
	}
$$

# Импульсная характеристика
Найдём реакцию системы на дельта функцию
![[Pasted image 20250126141925.png]]
В таком случае мы получим тот же самый H(z) то есть передаточную функцию и если мы возьмём обратное преобразование Лапласа от передаточной функции то мы получаем функцию которая показывает, как будет меняться сигнал во временной области. То есть мы получили характеристику системы во временной области, что как мы помним из [фильтров](Фильтрация%20и%20свёртка#FIR%20фильтр%20и%20знакомство%20со%20свёрткой) называется импульсной характеристикой
Импульсная характеристика позволяет определить динамические свойства системы, такие как частотные характеристики, время реакции и устойчивость

# Стабильность и передаточная функция
Мы можем разбить нашу передаточную функцию на полиномы
$$
	W(z)
=
	\frac{Y(s)}{X(s)}
=
	\frac
	{
		b_2+b_1z^{-1}+b_0z^{-2}
	}
	{
		a_2+a_1z^{-1}+a_0z^{-2}
	}
=
	K
	\frac
	{
		(z-{\color{orange} z_{z1}}) (z-{\color{orange} z_{z2}})
	}
	{
		(z-{\color{pink} z_{p1}}) (z-{\color{pink} z_{p2}})
	}
=
	A
	+
	\frac{B}
	{
		(z-{\color{pink} z_{p1}})
	}
	+
	\frac{C}
	{
		( z-{\color{pink} z_{p2}} )
	}
$$

Оранжевым выделены нули передаточной функции то есть если $z=z_{z1}$ или $z=z_{z2}$ то $W(z)=0$
Розовым же выделены полюса передаточной функции то есть если $s=z_{p1}$ или $s=z_{p2}$ то $\lim_{z\to z_{p1}}W(z)=\infty$
Давайте возьмём обратное z преобразование
Для этого нам понадобится формулы
![[Pasted image 20250126142715.png]]
Но у нас не получается пока так просто преобразовать, поэтому нам нужно изменить эту функцию
$$
	W(z)
=
=
	A
	+
	\frac{B}
	{
		(z-{\color{pink} z_{p1}})
	}
	+
	\frac{C}
	{
		( z-{\color{pink} z_{p2}} )
	}
=
	A
	+
	\frac{Bz^{-1}}
	{
		(1-{\color{pink} z_{p1}}z^{-1})
	}
	+
	\frac{Cz^{-1}}
	{
		( 1-{\color{pink} z_{p2}z^{-1}} )
	}
$$
В таком случае у нас получится
$w[n]=A\delta(t)+B(z_{p1})^{n-1}1[n-1]+C(z_{p2})^{n-1}1[n-1]$
И теперь вспоминай [[Фильтрация и свёртка#Пример IIR]] там мы узнали что если у нас есть коэффициент $a^{n}$ в импульсной характеристике то если a<0 то наша импульсная характеристика будет стремится к 0 и соответственно будет стабильной, но если a>1 то наша импульсная характеристика будет стремится к $\infty$ а значит будет нестабильной
![[Pasted image 20250126143409.png|363]] ![[Pasted image 20250126143259.png|300]]

Смотрите у нас получились почти такая же функция $a^n$ только $z_{pk}^{n-1}$ что по факту одно и то же, какая разница на сколько позже будет появляться эта функция в графике. Поэтому нам важно чтобы $|z_{pk}|$ были меньше 1 чтобы импульсная характеристика стремилась к 0 и соответственно система была устойчивой 
Почему именно модуль? Потому что у нас может быть также комплексным числом $z_{pk}=|z_{pk}|e^{jn\phi}$ однако нам не так важна фаза этого числа, она не влияет на стабильность, а нам важна амплитуда, именно она влияет на стабильность
> [!tldr] 
>  $|z_{pk}|<0 \implies$ система стабильна
>  ![[Pasted image 20250126144434.png]]
>  Да тут уже появляются отличия с [непрерывными системами](Передаточная%20функция#Стабильность%20и%20передаточная%20функция) как раз таки из-за того что нам нужно чтобы модуль был меньше 1, то из-за этого модуля у нас появляется такой "кружок стабильности™" но смысл точно такой же. Если хоть 1 полюс выйдет за этот круг то система будет нестабильной

# Примерное понимание АЧХ по передаточной функции
C учётом полученной информации про нули и полюса передаточной функции инженеры научились получать примерное представление о том какая будет частотная характеристика системы.
Да конечно мы можем заменить в передаточной функции z на $e^{j\omega}$ и таким образом получить частотную характеристику а затем взять её модуль чтобы получить АЧХ. 
$$
H(z)
=
	\frac{1-z^{-1}}{1+\frac{1}{2}z^{-1}} 
\xrightarrow{z=e^{j\Omega}} 
	\frac{1-e^{-j\Omega}}{1+\frac{1}{2}e^{-j\Omega}}
$$
То есть да так мы получили частотную характеристику которую можно засунуть в матлаб и получить все нужные нам графики. Но я покажу способ который хоть и не прям точный, но зато быстрый и даёт представления о системе в общем
Для этого нам надо перевести в форму для нахождение нулей и полюсов как я показывал в [[Z-преобразование#Передаточная функция дискретной системы]]
$$H(z)= \frac{z-1}{z+\frac{1}{2}} $$
Нули $z_{z1}=1$
Полюса $z_{p1}=-\frac{1}{2}$
Смотри из базовой матеши мы помним что нули это значение x функции когда она переходит в 0, а полюса это когда функция стремится к $\infty$
Мы всё это можем нарисовать на комплексном графике, который также называется z-плоскость, потому что на оси абсцисс у нас действительная часть z а на оси ординат мнимая часть z
![[Pasted image 20250126145605.png]]
Как мы видим точки полюсов и нулей не выходят за круг
Но что за круги? Ну у нас же ещё должна быть ось H(z) которая должна идти вверх, то есть у нас по факту 3д график, но мы показываем только оси Im(z) и Re(z).
И смотри <span style="color: blue">синяя</span> показывает впадины до 0, а <span style="color: red">красная</span>  показывает выпуклости до $\infty$.
