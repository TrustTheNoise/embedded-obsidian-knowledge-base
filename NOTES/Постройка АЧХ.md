# Постройка АЧХ
Пример постройки АЧХ для RC фильтра высоких частот
$$H (omega)=frac(j omega R C, 1+j omega R C)$$
У которого частота среза $f_(c)=5 0 0$Hz
```matlab
% Задаем частоту среза
fc = 500; % Частота среза 500 Гц
% Определяем параметры
R = 1; % Ом, произвольное значение
C = 1 / (2 * pi * R * fc); % Фарады, расчет из частоты среза
% Создаем частотный диапазон
f = linspace(0, 2000, 1000); % Частоты от 0 до 2000 Гц
omega = 2 * pi * f; % Преобразование частоты в угловую частоту

% Определяем передаточную функцию H(omega)
H = (j * omega * R * C) ./ (1 + j * omega * R * C);

% Вычисляем амплитудно-частотную характеристику (модуль H)
H_abs = abs(H);

% Строим график
figure;
plot(f, H_abs);
xlabel('Частота (Гц)');
ylabel('|H(\omega)|');
title('АЧХ фильтра');
grid on;
```
