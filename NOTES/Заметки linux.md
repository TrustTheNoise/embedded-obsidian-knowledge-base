
- Если нужно чтобы какой-то демон запускался на автостарте то нужно написать в hyprland.conf `exec-once = <name_prog>`
- У расширений для vscode оказывается свой вывод в Output. Output это вкладка которая находится в том же окошке что и терминал. Если перейти во вкладку Output то справа сверху будет выбор чей output будет выводится, там есть выводы для расширений. Так что через это можно узнать есть ли какая-то проблема
