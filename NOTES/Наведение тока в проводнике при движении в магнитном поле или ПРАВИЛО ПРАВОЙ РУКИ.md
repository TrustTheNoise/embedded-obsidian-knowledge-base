# Правило правой руки
Если ладонь обращена к северному полюсу(стрелки магнитного потока упираются в ладонь), а большой палец направлен по движению проводника, то 4 пальца будут показывать направление наведённой эдс в проводнике.
![[th-1771338516.jpg]]
Объяснение такому явлению в [[Электромагнитная индукция]]

У такого наведения есть формула
$$E=BVlcos(\alpha)$$
Выведем эту формулу. Для начала вспоминаем [[Закон Фарадея(эдс от магнитного потока)]]
$$E=\frac{d\Phi}{dt}$$
Теперь нам нужно вспомнить [формулу для магнитного потока](Магнитный%20поток) 
$$\Phi=BS$$
Теперь следите за финтами ушей. Если мы возьмем проводник с длинной l и подвигаем его на расстояние x, то по факту магнитные линии должны пройти через площадь равную S=xl, по сему 
$$\Phi=Bxl$$
Итак подставляем в формулу для наведенной эдс
$$E=\frac{d(Bxl)}{dt}=Bl\frac{dx}{dt}=BlV$$
