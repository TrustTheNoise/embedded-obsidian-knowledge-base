# Фильтрация в частотной области

Итак мы хотим создать фильтр нашего [[ЛЧМ сигнал(chirp signal)| chirp сигнала]] который будет пропускать только сигнал с частотами от 280 до 450 Гц. При этом частота дискретизации $f_s=1000 \text{ Гц}$

```MATLAB
% частота дискретизации
fs = 1000;
% время за которой изменяется частота сигнала
T = 0.5;
% период дескритезации
Ts = 1/fs;
% собственно говоря временные отсчёты графика от 0 до T
t = 0:Ts:T;

% тут мы как раз задаём функцию изменения частоты от времени
% тут мы изменяем частоту от 0 до fs/4 герц за время T
f = t*fs/(4*T);

% ну и сам сигнал создаём
x = sin(2*pi*f.*t);

% находим спектр нашего синала через преобразование фурье
X = fft(x);

% создаём вектор H размером как наш сигнал x и заполяем его нулями
H = zeros(1, length(x));
% выбираем частоты промежуток между которыми будет пропускать фильтр
from_frequency_pass = 280;
to_frequency_pass = 450;
% теперь нам нужно понять к какому номеру отсчёта относится
% нужные нам частоты, для этого мы используем выражение k/N = f/fs
% ну и нам нужно округлить это всё т.к. нам не нужны дробные числы
k1 = round( from_frequency_pass/fs*length(x) );
k2 = round( to_frequency_pass/fs*length(x) );

% заполняем нужные нам элементы в фильтре H единицами
% помним при этом о симметрии спектра для целых сигналов
% и о том что в матлабе первый элемент находится на x(1) (disgusting)
H(k1+1:k2+1) = 1;
H(length(x)-k2+1:length(x)-k1+1) = 1;

% собственно говоря перемножаем начальный спектр и фильтр
Y = X.*H;
% и находим обратное преобразование Фурье для полученного спектра
y = ifft(Y);
```

```MATLAB
subplot(325);

```

# Фильтрация в временной области
Итак у нас есть фильтр в временной области(ну т.к. у нас всё таки дискретные сигналы так или иначе но похуй (T-8)) h = `[1 -1]` который в частотной области представляет собой фильтр низких частот и усилитель высоких частот

*свёртка для данного сигнала*
Ну и мы будем фильтровать [[ЛЧМ сигнал(chirp signal)| chirp сигнал]] из прошлого примера.
```MATLAB
% частота дискретизации
fs = 1000;
% время за которой изменяется частота сигнала
T = 0.5;
% период дескритезации
Ts = 1/fs;
% собственно говоря временные отсчёты графика от 0 до T
t = 0:Ts:T;

% тут мы как раз задаём функцию изменения частоты от времени
% тут мы изменяем частоту от 0 до fs/4 герц за время T
f = t*fs/(4*T);

% ну и сам сигнал создаём
x = sin(2*pi*f.*t);

% находим спектр нашего синала через преобразование фурье
X = fft(x);
```
И после этого начинаются изменения 
```MATLAB
% наш фильтр в верменной области
h = [1 -1];
% свёртка нашего сигнала с фильтром
y = conv(x, h);
```
