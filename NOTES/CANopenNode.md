# Что это такое
CANopenNode это реализация стандарта [[CANopen]] в виде **открытой библиотеки на C**, которую можно встроить в микроконтроллер. При этом сам CANopenNode не содержит драйверов под конкретную платформу. То есть он сам не управляет CAN-контроллером он лишь даёт реализацию логики для CANopen. Для каждого типа процессора или ОС требуется написать «посредника» — драйвер, который свяжет ядро с конкретным CAN-аппаратным интерфейсом.
